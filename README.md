# peppol_mod
Peppol module voor Dollibarr software. 
This module adds an extra button to the invoice page, you are currently only able to download the invoice in the correct format that has been used by Peppol.

If you download this module, upload the folder to the "custom" folder in Dolibarr and then activate the module.

(Module is not finished - but allready works for downloading the XML files).
